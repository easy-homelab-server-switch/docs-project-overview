## Requirements
Before starting, make sure you have access to the following components and tools.


### Accounts
- MQTT broker
- Cloudflare account
- GitHub account


### Hardware
- Linux server
- Linux VM
- ESP32 microcontroller
- Client device
  - PC with Windows
  - **(optional)** Android phone


### Software
Depending on which components you deploy, you may need the following tools:
